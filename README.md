# מיזם גיאומטריה עם בינה מלאכותית - מחוז ירושלים

## סקירה
אתר מיזם גיאומטריה ובינה מלאכותית עבור מחוז ירושלים והעיר ירושלים מנח"י.
המיזם משלב מתמטיקה, יצירתיות וטכנולוגיית AI ליצירת חידות ויזואליות מתמטיות.

## מטרות המיזם
- העמקת ההבנה של מושגים ומשפטים גיאומטריים
- פיתוח חשיבה יצירתית וויזואלית
- טיפוח מיומנויות שימוש בבינה מלאכותית
- עידוד למידה חווייתית ושיתופית
- חיזוק הקשר בין מתמטיקה לאמנות ויצירה

## תכונות האתר
✅ **RTL מלא** - תמיכה מלאה בעברית  
✅ **עיצוב פרימיום** - כחול כהה, לבן וזהב  
✅ **אנימציות עדינות** - glow, pulse, hover effects  
✅ **כפתור הרשמה 3D** - עיצוב תלת-מימדי  
✅ **הטמעות** - YouTube ו-Canva אמיתיות  
✅ **רספונסיבי** - מתאים למובייל

## טכנולוגיות
- HTML5 + CSS3  
- JavaScript (Vanilla)  
- Google Fonts (Heebo, Assistant)  
- Responsive Design

לשאלות נוספות, ניתן לפנות למדריכות המתמטיקה והתקשוב מחוז ירושלים.
